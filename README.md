# myfstProject

## 猜數字遊戲

這是一個簡單的猜數字遊戲，使用 Python 開發。

### 遊戲規則

1. 程式隨機產生一個 1 到 100 的整數
2. 玩家每次輸入猜測數字
3. 程式會提示：
   - **小了** - 猜測的數字小於答案
   - **大了** - 猜測的數字大於答案
   - **恭喜通過！** - 猜對了！
4. 程式會記錄並顯示猜測次數

### 如何執行

```bash
python3 Gn.py
```

### 遊戲玩法

- 輸入 1 到 100 之間的數字進行猜測
- 輸入 'exit' 可以離開遊戲
- 遊戲會顯示你的猜測次數，幫助你追蹤進步

### 範例

```
歡迎來到猜數字遊戲！我已經選好了一個1到100之間的數字。
請輸入你的猜測（或輸入 'exit' 離開遊戲）：50
小了
請輸入你的猜測（或輸入 'exit' 離開遊戲）：75
大了
請輸入你的猜測（或輸入 'exit' 離開遊戲）：65
恭喜通過！數字就是 65。你總共猜了 3 次。
```
